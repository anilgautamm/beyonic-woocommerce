This Plugin integrates Beyonic Payments with your woocommerce website.

== Description ==

Beyonic is a technology company that aims to eliminate the rampant use of cash in emerging markets. Beyonic’s partnerships with mobile network operators and digital financial service providers across Africa enable businesses to quickly deploy and manage 2-way mobile money payments. Via secure, web-based services and robust APIs, our customers access enterprise level payee and workflow management to reduce fraud and increase transparency across their operations.

Pay people using mobile money.
Some of the world’s most innovative teams trust Beyonic to help transform their payments.


== Installation ==


Install the beyonic plugin from your WordPress Dashboard, under Plugins → Add New section.
Activate the plugin
Configure the options in the WooCommerce → Settings → Checkout screen. This includes the API keys you got from your Beyonic Dashboard


== Frequently Asked Questions ==

= What version of PHP is required for this plugin to work? =
